# chess
Minimal Chess game using zig 0.15 and raylib.
Here are the links for assets:
https://unknuffig.itch.io/2d-chess-pices
https://commons.wikimedia.org/wiki/Category:PNG_chess_pieces/Standard_transparent

# Installation
It should just run with a simple `zig build run`
